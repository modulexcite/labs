Boston University Bionic Pancreas UI prototype

A prototype user interface to the Bionic Pancreas from Boston University. Consists of the following three components:

  • Offline Web App interface

    Use Chrome Beta to add the app to home screen. Need help?. Follow these steps.

  • Web App interface

    Access the protype from any web browser.

  • Remote control

    Control the levels of various systems in the insulin pump, and see the changes reflected in the web app
